Team Task Manager Demo Video Script

Suggested length: 3 to 5 minutes.

1. Introduction
Say:
"This is Team Task Manager, a full-stack MERN web application built with React, Node.js, Express, MongoDB, Mongoose, JWT authentication, and role-based access control."

Show:
- GitHub repository homepage
- README.md file
- Folder structure with frontend and backend folders

2. Authentication
Show:
- Login page
- Signup page

Say:
"Users can create an account or log in. Passwords are hashed using bcrypt, and authenticated requests use JWT tokens."

Demo login:
Email: admin@teamtask.local
Password: Admin12345!

3. Dashboard
Show:
- Dashboard after login
- Task summary cards

Say:
"The dashboard shows tasks assigned to the logged-in user, including completed, pending, and overdue counts. Overdue tasks are highlighted."

4. Role-Based Access Control
Say:
"The app supports two roles: Admin and Member. Admin users can create projects, manage team members, and create or delete tasks. Member users can only view assigned projects and update their assigned task status."

5. Project Management
Show:
- Projects page
- Create a new project
- Select team members
- Open project detail page

Say:
"Admins can create, update, and delete projects. Each project stores a name, description, team members, and the admin who created it."

6. Task Management
Show:
- Create a task inside a project
- Assign task to a user
- Set status and due date
- Filter by status or assignee
- Update task status

Say:
"Tasks belong to projects and are assigned to team members. The app supports Todo, In Progress, and Done statuses, due dates, and filtering."

7. Backend and Database
Show:
- backend/src folders: models, routes, controllers, middleware, validators
- MongoDB Atlas dashboard if allowed

Say:
"The backend is organized using controllers, routes, models, middleware, and validators. MongoDB relationships are handled with ObjectId references between users, projects, and tasks."

8. Deployment
Show:
- railway.json
- package.json scripts

Say:
"The app is deployment-ready for Railway. The root build command installs backend and frontend dependencies and builds the React app. The start command runs the Express server, which serves both the API and frontend."

9. Closing
Say:
"This completes the demo of Team Task Manager: a production-ready team project and task management application with authentication, authorization, dashboard analytics, and MongoDB persistence."
